# Project A1

A1 is my personal AI assistant project that I am building from scratch to learn software engineering, automation, and AI development.

## Current Features

* Add tasks
* View tasks
* Complete tasks
* Persistent task storage using JSON
* Command-line based interface

## Project Structure

```text
A1
├── data
│   └── tasks.json
├── modules
│   └── task_manager.py
└── main.py
```

## Tech Stack

* Python
* JSON
